# PaperWork

A Hugo theme forked from [PaperMod](https://github.com/adityatelange/hugo-PaperMod), with some features and customizations added but still keeps simplicity of original theme.

## Features and Changes

- Image Zoom
- Read also
- *Upcoming* Box Info
